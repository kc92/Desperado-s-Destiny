# Crafting Specialization Paths - Quick Reference

## Overview
- **Total Paths**: 18 (3 per profession)
- **Requirement**: Level 50 in profession
- **Cost**: 5000 gold
- **Rule**: One specialization per profession
- **Mastery**: 0-100% progress toward legendary rewards

## Gameplay Strategy

### Early Game (Level 1-49)
- Train profession skills to level 50
- Accumulate 5000 gold per profession you want to specialize
- Research which specializations fit your playstyle

### Mid Game (Level 50+)
- Choose your first specialization
- Craft unique recipes to gain mastery progress
- Build toward 100% mastery

### End Game (Mastery 100%)
- Unlock mythic recipes
- Craft the most powerful items in the game
- Display mastery title
- Consider specializing in additional professions

## Specialization Synergies

### Combat Build
- Weaponsmith (damage) + Ammunition Expert (special ammo) + Apothecary (combat buffs)

### Stealth Build
- Leather Armorsmith (stealth) + Poisoner (deadly toxins) + Disguise Master (infiltration)

### Social Build
- Fashion Designer (reputation) + Brewer (social buffs) + Disguise Master (versatility)

### Survival Build
- Utility Tailor (weather protection) + Preserver (endless food) + Repair Specialist (maintain gear)

### Economy Build
- Bag Maker (carry more) + Chef (high-value food) + Weaponsmith (legendary weapons to sell)
