- **Position**: Fixed full-screen

## ğŸ’¡ Did You Know?

### Fun Facts About the System

1. **Smooth Transitions**: The 3-second transition was chosen to match a natural eye adjustment time

2. **Color Science**: Blue tints at night trigger melatonin in real life - we use very subtle blue to suggest danger without affecting gameplay

3. **Western Authenticity**: Dusk colors match real Western desert sunsets (orange to purple gradient)

4. **Performance**: The overlay uses CSS only - zero JavaScript animation cost = 60 FPS always

5. **Accessibility**: Screen readers announce time changes, colorblind modes coming in Phase 2
