## Usage Examples

### Finding Affordable Properties
```typescript
import { getAffordableUrbanProperties } from './urbanPropertyIndex';

const player = {
  level: 10,
  gold: 3000,
  skills: { blacksmithing: 15, leatherworking: 8 }
};

const affordable = getAffordableUrbanProperties(
  player.level,
  player.gold,
  player.skills
);
// Returns 10 properties player can afford
```

### Getting Properties by Location
```typescript
import { getUrbanPropertiesByLocation } from './urbanPropertyIndex';

const redGulchProps = getUrbanPropertiesByLocation('Red Gulch');
// Returns 7 properties in Red Gulch
```

### Calculating Saloon Profit
```typescript
import { calculateWeeklySaloonProfit } from './saloons';

const profit = calculateWeeklySaloonProfit(SALOONS.lucky_strike_saloon);
// ~1,705g/week profit
```

### Checking Purchase Eligibility
```typescript
import { canPlayerPurchase } from './urbanPropertyIndex';

const player = {
  level: 14,
  gold: 3500,
  reputation: { frontera: 250 },
  skills: { gunsmithing: 25 }
};

const result = canPlayerPurchase(SALOONS.casa_de_placer, player);
// { canPurchase: true, reasons: [] }
```

## Notes

- All prices in gold pieces
- Weekly costs include tax + upkeep (rent if applicable)
- Daily revenue estimates assume good management
- Workshop bonuses apply to crafted item quality/speed
- Facility tiers: 1 (basic) to 5 (legendary)
- Location multipliers affect customer willingness to pay premium prices
